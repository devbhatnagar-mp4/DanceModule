# Dance Practice Chrome Extension
A Chrome extension built to make dance practice smoother.

## Features
- Mirror YouTube videos
- Loop specific sections (hh:mm:ss - hh:mm:ss)
- Speed control
- Works in fullscreen

## Why I built this
Learning/choreographing using YouTube felt unnatural ,constant rewinding, mirroring mentally.

So I built a tool to fix that.

## Tech Stack
HTML, CSS, JavaScript, Chrome Extensions API, DOM manipulation.
